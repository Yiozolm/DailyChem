Q:你其实可以把自己工作流总结一下，我可以看看怎么用ai代理一些。

A:我感觉最重要的就是用ai解析谱。

Q:你光说这个我也不知道咋做啊?最好就是把你科研要用到的，都说一下。

A:主要我感觉用ai解谱是个目前不太好做到的事儿。就是根据目标化合物分子的结构判断出每个原子的化学位移、以及耦合裂分，然后根据预测的谱和实际的谱去对应，能对得上就OK，对不上就错。

Q：这个是通过软件计算的？这个任务里面哪个是最费时间的？

A：是有个标准的 就是不同化学环境的原子有一个大致的化学位移范围。类似于这样，但这只是氢谱，核磁共振还有碳谱 氟谱 磷谱之类的，然后除了核磁共振 还有红外 质谱 紫外。

Q：这个实际的谱就是你做实验得到的化合物的谱？那其实就是，你先设计一个化合物结构，计算出理论上的性质，再从你做实验得出的化合物实际结果，去比较对吧。

A：简单来说就是这样的。或者如果要再简单点 就是我每个峰去找对应的归属，我看着实际打出来的谱 每个峰都去找对应。

Q：那这实际上是个图像比对的事情呗。还有其他的琐碎的事吗？比方说数据记录啥的。

A：数据记录吗？这个倒是，做实验要做实验记录的，我最不喜欢干的就是实验记录，因为我做实验究极大量。就是我会用一个软件叫做chemdraw，这个软件是可以预测化合物的谱的，可以用于对比。但它不一定准，所以是用于辅助对比。
